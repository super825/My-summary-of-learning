# idea常用快捷键

| 快捷键       | 功能                     |
| ------------ | ------------------------ |
| ctrl+Z       | 撤消                     |
| ctrl+shift+Z | 重做                     |
| ctrl+F       | 当前文件检索             |
| ctrl+shift+F | 全文检索                 |
| ctrl+R       | 当前文件查找替换         |
| ctrl+shift+R | 全文检索替换             |
| double shift | 全局查找                 |
| alt+F7       | 查找当前方法调用地方     |
| ctrl+alt+B   | 快速定位当前方法的实现   |
| alt+insert   | 代码生成菜单             |
| ctrl+/       | 行注释                   |
| ctrl+shift+/ | 代码块注释               |
| ctrl+alt+L   | 格式化                   |
| ctrl+alt+O   | 优化import               |
| alt+enter    | 快速修改错误             |
| shift+F6     | 文件重命名               |
| ctrl+D       | 复制当前行并粘贴到下一行 |
| ctrl+Y       | 删除当前行               |
